World Computer Sculpture Garden



A contract show organized by 0xfff
with special thanks to sssluke and 113





~ I N V I T A T I O N ~


I invite you into the garden of many running sculptures – an open ground for all to visit. Feel encouraged to move forward, to go beyond the white cube VR-replica spaces, and ignore the stratifying forces that seek to mummify New Art. Dare to disregard their pleas for commodification. Let us re-consider the durable, distributed computational substrate as a canvas for aesthetic, poetic, philosophical, and practical explorations. Open to surprise. Open to delight. Purposefully non-standard. Come and display your creations in the World Computer Sculpture Garden ⚘





~ C O N C E P T ~


"World Computer Sculpture Garden" is a contract show on Ethereum Mainnet scheduled for October 2024. Participating artists exhibit their works by submitting a contract address that is included in the show's main contract. The only requirement for the pieces is a contract address that implements a minimal set of fields. The rest is up to the artists.





~ T I M E L I N E ~


. Confirmation of participation by July 31st, 2024
. Round table #1 with artists in August (optional)
. Round table #2 with artists in September (optional)
. Testnet deployment of artworks by October 1st, 2024
. Deployment of artworks by October 14th, 2024
. Opening of show October 30th, 2024 (Vernissage in Discord)





~ F I N A N C I A L S ~


I am working on securing funding to pay participating artists a stipend. At minimum deployment costs will be covered. Sponsors and patrons please contact 0xfff@protonmail.com for inquiries.





~ S A L E S ~


This is not a sales show. There is no requirement or expectation for the artists to sell their works or for the works to be ownable in the first place. Artists retain full rights to their works.





~ T E C H N I C A L S ~


The show contract will pull the fields from all works and generate an html website which allows for dynamic responses by the individual works contracts. There is no expectation of any visual outcome. Please adhere to the following interface for your contract (subject to change). 
interface Sculpture {

    function title() external view returns (string memory);

    function authors() external view returns (string[] memory);

    function addresses() external view returns (address[] memory);

    function urls() external view returns (string[] memory);

    function text() external view returns (string memory);

    // NOTE: These fields will be included as HTML in the
    // contract-generated exhibition page. All fields can
    // return empty values if not applicable.

}

0xfff will be available for support and guidance throughout the process of developing your piece.
